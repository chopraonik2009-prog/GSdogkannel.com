# GS Dog Kannel

Welcome to the official website repository for **GS Dog Kannel**, Punjab's premium Kennel Club of India (KCI) registered breeder.

## Features
- **Modern UI/UX**: Designed with sleek, premium aesthetics featuring glassmorphism, dynamic animations, and responsive layouts.
- **Mobile-Friendly**: Fully responsive and optimized for seamless viewing on phones, tablets, and desktop devices.
- **KCI Certified**: Built-in verification links and premium branding representing championship lineage breeding.
- **Integrated Contact Methods**: Direct click-to-call, WhatsApp integration, and consultation booking forms.
- **Compliance Ready**: Includes standard Privacy Policy, Terms & Conditions (Anti-Spam Policy), and Legal Disclaimers.

## Project Structure
- `index.html` - The main landing page.
- `about.html` - Information about the history and legacy of GS Dog Kannel.
- `breeds.html` - A gallery and details of the breeds available (German Shepherds, Rottweilers, etc.).
- `appointments.html` - The consultation and puppy inquiry booking forms.
- `why-choose-us.html` - The kennel's unique selling points and KCI standards.
- `privacy.html`, `terms.html`, `disclaimer.html` - Standard legal compliance pages.
- `index.css` - The master stylesheet containing all design tokens, animations, and responsive rules.
- `index.js` - JavaScript handling the dynamic mobile menu, smooth scrolling, and form modal interactions.
- `assets/` - Directory containing all optimized images and visual assets.

## Deployment Instructions (GitHub Pages)

This website is a static site (HTML, CSS, JS) which makes it incredibly easy to host for free using **GitHub Pages**.

1. Upload all the files in this folder to your GitHub repository.
2. In your GitHub repository, click on the **Settings** tab.
3. On the left sidebar, scroll down and click on **Pages**.
4. Under "Build and deployment", set the **Source** to `Deploy from a branch`.
5. Under "Branch", select your main branch (usually `main` or `master`), select the `/ (root)` folder, and click **Save**.
6. Wait a couple of minutes! GitHub will generate a live URL for your website (typically `https://[your-username].github.io/[repository-name]/`).
